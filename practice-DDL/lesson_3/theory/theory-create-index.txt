Команда CREATE INDEX используется для создания индекса на одном
или нескольких столбцах таблицы базы данных.

Индексы улучшают производительность выполнения запросов,
позволяя быстрее находить и извлекать данные из таблицы.

Основной синтаксис команды CREATE INDEX выглядит следующим образом:
```sql
CREATE INDEX index_name
ON table_name (column1, column2, ...);
```
Где:
- `CREATE INDEX`: ключевые слова, указывающие на создание индекса.
- `index_name`: название индекса, которое вы хотите задать.
- `table_name`: название таблицы, на которую будет создан индекс.
- `(column1, column2, ...)`: список столбцов, на которых будет создан индекс.


Пример команды CREATE INDEX:
```sql
CREATE INDEX idx_customers_last_name
ON customers (last_name);
```
В этом примере мы создаем индекс с названием "idx_customers_last_name"
на столбце "last_name" таблицы "customers". Индекс будет использоваться
для ускорения поиска и сортировки данных по этому столбцу.

Индексы могут быть созданы на одном или нескольких столбцах таблицы.
Они позволяют быстрее выполнять операции SELECT, JOIN и ORDER BY,
так как сокращают количество записей, которые необходимо просмотреть
при выполнении запросов.

Однако, создание индексов может занять некоторое время и увеличить
размер базы данных. Кроме того, индексы требуют обновления при
изменении данных в таблице, поэтому необходимо внимательно выбирать,
на каких столбцах создавать индексы и когда их создавать.

Команда CREATE INDEX позволяет создавать индексы в таблицах базы данных,
что повышает производительность выполнения запросов.

Она является одной из команд DDL (Data Definition Language) в SQL,
используемых для определения структуры и настроек базы данных.
