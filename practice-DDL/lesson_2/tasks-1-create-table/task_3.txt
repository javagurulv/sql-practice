Напишите SQL команду для создания в базе данных таблицы
зданий. Про здания нужно сохранять в базу данных:
- id (как primary key со стратегией генерации авто инкремент)
- название
- вместимость (измеряется в количестве человек).
Название должно быть обязательными, вместимость нет.
(воспользуйтесь SQL командами CREATE TABLE).
