Напишите SQL команды для создания в базе данных таблиц
книг, читателей и читательского билета.

Про книгу нужно сохранять в базу данных:
- название
- автора.

Про читателя нужно сохранять в базу данных:
- полное имя (имя + фамилия в одном поле)
- персональный код.

Про читательский билет нужно сохранять в базу данных:
- ссылку на книгу, которую читатель взял на чтение
- ссылку на читателя, кто взял книгу на чтение
- дату, когда читатель взял книгу на чтение
- дату, когда читатель вернул книгу в библиотеку.

Все поля должны быть обязательными.

(воспользуйтесь SQL командами CREATE TABLE).
