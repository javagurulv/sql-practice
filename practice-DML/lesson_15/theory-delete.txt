Команда SELECT DELETE используется для удаления одной или
нескольких строк из таблицы базы данных.

Она позволяет удалить записи, которые соответствуют заданным условиям.

Основной синтаксис команды DELETE выглядит следующим образом:

DELETE FROM table_name
WHERE condition;

Где:
- `DELETE FROM`: ключевое слово, указывающее на начало команды удаления данных.
- `table_name`: название таблицы, из которой вы хотите удалить строки.
- `WHERE condition`: условие, определяющее, какие строки следует удалить. Если условие не указано, будут удалены все строки таблицы.


Пример 1: Простое использование команды DELETE:

DELETE FROM customers
WHERE customer_id = 1001;

В этом примере мы удаляем строку из таблицы "customers",
где значение столбца "customer_id" равно 1001.


Пример 2: Использование команды DELETE с условием:

DELETE FROM orders
WHERE order_date < '2022-01-01';

В этом примере мы удаляем все строки из таблицы "orders",
где значение столбца "order_date" меньше заданной даты '2022-01-01'.


Команда DELETE позволяет удалять строки из таблицы базы данных,
соответствующие заданным условиям. Она полезна для удаления
ненужных данных или исправления ошибок в таблице.

Важно быть осторожным при использовании команды DELETE,
так как она приводит к фактическому удалению данных из таблицы,
и удаленные данные не могут быть восстановлены без резервной копии.
