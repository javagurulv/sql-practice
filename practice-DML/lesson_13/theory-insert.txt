Команда SELECT INSERT, или просто INSERT, используется для
вставки новых строк данных в таблицу базы данных.

Она позволяет добавить новые записи в таблицу с указанными
значениями для каждого столбца.

Основной синтаксис команды INSERT выглядит следующим образом:

INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...);

Где:
- `INSERT INTO`: ключевое слово, указывающее на начало команды вставки данных.
- `table_name`: название таблицы, в которую вы хотите вставить новые строки.
- `(column1, column2, ...)`: список столбцов таблицы, для которых вы хотите указать значения. (Необязательно, если вставляются значения для всех столбцов таблицы.)
- `VALUES (value1, value2, ...)`: значения, которые вы хотите вставить для каждого столбца таблицы в указанном порядке.


Пример 1: Простое использование команды INSERT:

INSERT INTO employees (first_name, last_name, age)
VALUES ('John', 'Doe', 30);

В этом примере мы вставляем новую строку в таблицу "employees"
со значениями для столбцов "first_name", "last_name" и "age".
Значения 'John', 'Doe' и 30 вставляются соответственно.


Пример 2: Вставка нескольких строк с помощью команды INSERT:

INSERT INTO employees (first_name, last_name, age)
VALUES ('John', 'Doe', 30),
       ('Jane', 'Smith', 25),
       ('Mark', 'Johnson', 35);

В этом примере мы вставляем три новые строки в таблицу "employees"
с указанными значениями для столбцов "first_name", "last_name" и "age".
Каждая строка представлена отдельным списком значений внутри оператора
VALUES.


Команда INSERT позволяет добавлять новые строки данных в таблицу
базы данных.
