Команда INSERT в SQL

Команда INSERT, или просто INSERT, используется для вставки новых строк данных в таблицу базы данных.

Пример 1: Простое использование команды INSERT

INSERT INTO employees (first_name, last_name, age)
VALUES ('John', 'Doe', 30);

В этом примере мы вставляем новую строку в таблицу "employees" с указанными значениями для столбцов "first_name", "last_name" и "age". Значения 'John', 'Doe' и 30 вставляются соответственно.

Пример 2: Вставка нескольких строк с помощью команды INSERT

INSERT INTO employees (first_name, last_name, age)
VALUES ('John', 'Doe', 30),
       ('Jane', 'Smith', 25),
       ('Mark', 'Johnson', 35);

В этом примере мы вставляем три новые строки в таблицу "employees" с указанными значениями для столбцов "first_name", "last_name" и "age". Каждая строка представлена отдельным списком значений внутри оператора VALUES.

Заключение

Команда INSERT позволяет добавлять новые строки данных в таблицу базы данных.
