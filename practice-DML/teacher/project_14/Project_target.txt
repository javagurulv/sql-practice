Цель данного проекта заключается в изучении использования
оператора UPDATE в языке SQL для изменения данных в таблице базы данных.

Ключевые моменты для достижения этой цели включают:

1. Понимание оператора UPDATE: Изучите синтаксис оператора UPDATE
и его функциональность для изменения данных в таблице. Понимание
различных частей оператора UPDATE, таких, как имя таблицы, столбцы
и условия, поможет вам правильно формировать запросы на обновление данных.

2. Обновление значений в одной строке: Практикуйтесь в написании
запросов UPDATE для изменения значений в одной строке таблицы.
Узнайте, как указывать имя таблицы, столбцы и новые значения,
чтобы правильно обновить данные.

3. Обновление значений в нескольких строках: Изучите, как обновлять
значения в нескольких строках таблицы одновременно с помощью оператора
UPDATE. Практикуйтесь в написании запросов, которые обновляют данные
для нескольких строк, используя один оператор UPDATE, что может быть
эффективным при массовом изменении данных.

4. Учет условий обновления: Изучите, как использовать условия в
операторе UPDATE для определения, какие строки будут обновлены.
Практикуйтесь в написании запросов, которые применяют условия для
обновления только определенных строк, в соответствии с вашими требованиями.

5. Расширение знаний: Обновление данных с использованием подзапросов:
Изучите, как использовать подзапросы в операторе UPDATE для обновления
данных на основе значений из других таблиц или выборок. Практикуйтесь
в написании запросов, которые извлекают данные из других таблиц и
используют их для обновления данных в целевой таблице.
