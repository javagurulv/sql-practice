Ниже приведена инструкция по установке MySQL Community Server
с использованием дефолтных настроек:

Шаг 1: Скачать MySQL Community Server
1. Перейдите на официальный сайт MySQL (https://www.mysql.com/downloads/)
и найдите раздел "MySQL Community Edition".
2. В разделе "MySQL Community (GPL) Downloads" выберите
"MySQL Community Server".
3. Выберите подходящую версию для вашей операционной системы
и нажмите на ссылку для загрузки.


Шаг 2: Запустить установку
1. После загрузки установочного файла MySQL Community Server запустите его.
2. Вас приветствует мастер установки. Нажмите "Next" (Далее), чтобы продолжить.


Шаг 3: Принять лицензионное соглашение
1. Ознакомьтесь с лицензионным соглашением MySQL Community Server.
2. Если вы согласны с условиями, выберите "I accept the terms in
the License Agreement" (Я принимаю условия Лицензионного соглашения)
и нажмите "Next" (Далее).


Шаг 4: Выбрать компоненты для установки
1. Вам будет предложено выбрать компоненты для установки.
2. Убедитесь, что выбран пункт "Server" (Сервер).
3. Нажмите "Next" (Далее).


Шаг 5: Выбрать настройки установки
1. Выберите опцию "Developer Default" (Разработчик, по умолчанию).
2. Убедитесь, что включена опция "Server Machine" (Серверная машина).
3. Нажмите "Next" (Далее).


Шаг 6: Выбрать тип настройки сервера
1. Выберите тип настройки сервера, который наилучшим образом
соответствует вашим потребностям.
2. Если вы не уверены, оставьте выбранную опцию
"Standalone MySQL Server / Classic MySQL Replication"
(Отдельный сервер MySQL / Классическая репликация MySQL).
3. Нажмите "Next" (Далее).


Шаг 7: Указать путь установки
1. Оставьте предложенный путь установки по умолчанию
или выберите другую папку для установки.
2. Нажмите "Next" (Далее).


Шаг 8: Настроить тип сетевого подключения
1. Выберите опцию "Config Type" (Тип конфигурации)
"Server Machine" (Серверная машина).
2. Нажмите "Next" (Далее).


Шаг 9: Установка
1. Нажмите "Install" (Установить), чтобы начать установку
MySQL Community Server с выбранными настройками.
2. Подождите, пока установка завершится.


Шаг 10: Завершение установки
1. Установка успешно завершена.
2. Убедитесь, что опция "Launch the MySQL Installer for Windows"
(Запустить программу установки MySQL для Windows) выбрана.
3. Нажмите "Finish" (Завершить).

Поздравляю! Вы установили MySQL Community Server с дефолтными настройками.
Теперь вы можете использовать его для разработки приложений или для других нужд.


После установки MySQL Community Server и запуска программы установки,
вам будет предложено запустить "MySQL Installer for Windows"
(если вы устанавливаете на Windows) или аналогичное средство управления
установкой для вашей операционной системы. В этой программе установки
будет доступна опция для настройки сервера и установки рутового пароля.

После выбора опции для настройки сервера, вам может быть предложено
ввести новый рутовый пароль или изменить существующий, если он уже
был установлен. Введите требуемый пароль в соответствующее поле и
продолжайте выполнение настроек сервера.

Важно выбрать надежный пароль для рутового пользователя,
поскольку это дает полный контроль над вашей базой данных MySQL.
Убедитесь, что пароль достаточно сложный и хорошо запоминается,
но не подвержен угадыванию.

После установки рутового пароля и завершения настройки сервера вы
сможете использовать его для доступа к MySQL и управления базами данных.
