Команда SELECT используется для выборки данных из базы данных.

В самом простом варианте она позволяет указать,
какие столбцы или поля следует выбрать из таблицы
находящейся в базе данных.

Самый простой синтаксис команды SELECT выглядит следующим образом:
```sql
SELECT column1, column2, ...
FROM table_name;
```

Основные элементы команды SELECT:

SELECT: ключевое слово, указывающее на начало команды выборки данных.

column1, column2, ...: названия столбцов или полей,
которые вы хотите выбрать из таблицы. Если вы хотите
выбрать все столбцы, можно использовать символ звездочки *.

FROM table_name: указывает таблицу или таблицу, из которой
следует выбрать данные.



Вот несколько примеров простых команд SELECT:

Пример 1. Пример выборки всех столбцов из таблицы "users":
```sql
SELECT * FROM users;
```

Пример 2. Пример выборки конкретных столбцов из таблицы "products":
```sql
SELECT product_name FROM products;
```

Пример 3. Пример выборки конкретных столбцов из таблицы "products":
```sql
SELECT product_name, price FROM products;
```
