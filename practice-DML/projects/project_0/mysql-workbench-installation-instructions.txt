Ниже приведена инструкция по установке MySQL Workbench
с использованием дефолтных настроек:

Шаг 1: Скачать MySQL Workbench
1. Перейдите на официальный сайт MySQL (https://www.mysql.com/downloads/)
и найдите раздел "MySQL Workbench".
2. В разделе "MySQL Workbench Downloads" выберите подходящую версию
для вашей операционной системы и нажмите на ссылку для загрузки.


Шаг 2: Запустить установку
1. После загрузки установочного файла MySQL Workbench запустите его.
2. Вас приветствует мастер установки. Нажмите "Next" (Далее),
чтобы продолжить.


Шаг 3: Принять лицензионное соглашение
1. Ознакомьтесь с лицензионным соглашением MySQL Workbench.
2. Если вы согласны с условиями, выберите "I accept the terms
in the License Agreement" (Я принимаю условия Лицензионного соглашения)
и нажмите "Next" (Далее).


Шаг 4: Выбрать папку установки
1. Оставьте предложенную папку установки по умолчанию или
выберите другую папку, если требуется.
2. Нажмите "Next" (Далее).


Шаг 5: Установка
1. Нажмите "Install" (Установить), чтобы начать установку
MySQL Workbench с выбранными настройками.
2. Подождите, пока установка завершится.


Шаг 6: Завершение установки
1. Установка успешно завершена.
2. Убедитесь, что опция "Launch MySQL Workbench"
(Запустить MySQL Workbench) выбрана.
3. Нажмите "Finish" (Завершить).


Поздравляю! Вы установили MySQL Workbench с дефолтными настройками.
Теперь вы можете использовать его для подключения к серверу MySQL,
создания и управления базами данных, выполнения запросов и многого другого.
