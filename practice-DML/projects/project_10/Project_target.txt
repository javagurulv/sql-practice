Цель данного проекта заключается в освоении оператора GROUP BY
и агрегатных функций в языке SQL для группировки данных и
выполнения вычислений над группами.

Ключевые моменты для достижения этой цели включают:

1. Понимание оператора GROUP BY: Изучите синтаксис оператора GROUP BY
и его роль в группировке данных. Понимание концепции группировки
и ее применения поможет вам правильно структурировать запросы
для анализа данных.

2. Группировка данных: Практикуйтесь в написании запросов с
использованием оператора GROUP BY для группировки данных по
одному или нескольким столбцам. Узнайте, как правильно указывать
столбцы для группировки, и как группировка влияет на результаты запроса.

3. Агрегатные функции: Изучите различные агрегатные функции,
такие как COUNT, SUM, AVG, MIN и MAX, и понимайте их использование
для выполнения вычислений над группами данных. Практикуйтесь в
использовании этих функций для получения сумм, средних значений,
количества элементов и других агрегированных результатов.

4. Использование агрегатных функций с GROUP BY: Практикуйтесь в
написании запросов, комбинирующих оператор GROUP BY и агрегатные
функции, для получения агрегированных результатов по группам данных.
Узнайте, как правильно структурировать запросы, чтобы получить
необходимые вычисления для каждой группы данных.

5. Условия фильтрации и агрегация: Изучите, как применять условия
фильтрации (с помощью оператора WHERE) вместе с GROUP BY и
агрегатными функциями для получения более специфических результатов.
Практикуйтесь в использовании фильтров для агрегированных данных.
