Цель данного проекта заключается в освоении использования
оператора IS NULL в языке SQL для работы с отсутствующими значениями.

Ключевые моменты для достижения этой цели включают:

1. Понимание концепции отсутствующих значений: Изучите, что
такое отсутствующие значения в базах данных и как они отличаются
от пустых строк или нулевых чисел. Разберитесь с причинами
возникновения отсутствующих значений в таблицах.

2. Ознакомление с оператором IS NULL: Изучите синтаксис оператора
IS NULL в SQL и понимание его использования для проверки,
содержат ли столбцы отсутствующие значения.

3. Фильтрация данных с использованием IS NULL: Практикуйтесь в
написании запросов, используя оператор IS NULL для фильтрации данных.
Узнайте, как выбрать строки, в которых определенные столбцы содержат
отсутствующие значения, и как комбинировать операторы IS NULL с
другими условиями.

4. Расширение знаний: После овладения оператором IS NULL,
исследуйте более сложные аспекты работы с нулевыми значениями,
такие как использование IS NOT NULL.
