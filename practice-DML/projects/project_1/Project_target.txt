Цель данного проекта познакомиться с SQL и освоить использование
оператора SELECT в языке SQL для выборки данных из таблицы без
применения условий.

Ключевые моменты для достижения этой цели включают:

1. Понимание оператора SELECT: Изучите синтаксис оператора SELECT
и его основные элементы, такие как список столбцов и имя таблицы.
Понимание этих элементов поможет вам правильно формировать запросы SELECT.

2. Выборка всех столбцов: Практикуйтесь в написании запросов SELECT,
выбирающих все столбцы из таблицы. Изучите, как указывать символ
звездочки (*) в качестве списка столбцов, чтобы выбрать все
доступные столбцы.

3. Использование именованных столбцов: Изучите, как указывать
конкретные имена столбцов в списке SELECT, чтобы выбрать только
необходимые столбцы из таблицы. Практикуйтесь в выборке конкретных
столбцов для более узкой и специфической выборки данных.
