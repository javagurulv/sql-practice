Команда SELECT UPDATE используется для изменения (обновления)
существующих строк данных в таблице базы данных.

Она позволяет изменить значения столбцов в одной или нескольких
строках таблицы согласно заданным условиям.

Основной синтаксис команды UPDATE выглядит следующим образом:
```sql
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
```
Где:
- `UPDATE`: ключевое слово, указывающее на начало команды обновления данных.
- `table_name`: название таблицы, в которой вы хотите изменить данные.
- `SET column1 = value1, column2 = value2, ...`: указывает столбцы и новые значения, которые вы хотите присвоить им.
- `WHERE condition`: условие, которое определяет, в каких строках таблицы следует выполнить обновление. Если условие не указано, все строки таблицы будут обновлены.


Пример 1: Простое использование команды UPDATE:
```sql
UPDATE employees
SET salary = 5000
WHERE department = 'IT';
```
В этом примере мы обновляем значение столбца "salary" на 5000 для
всех строк таблицы "employees", где значение столбца "department"
равно 'IT'.


Пример 2: Использование команды UPDATE с несколькими столбцами и условием:
```sql
UPDATE employees
SET salary = salary * 1.1, age = age + 1
WHERE department = 'Sales' AND salary < 5000;
```
В этом примере мы обновляем значения столбцов "salary" и "age" в
таблице "employees". Значение "salary" увеличивается на 10%
(умножается на 1.1), а значение "age" увеличивается на 1 для всех
строк, где значение столбца "department" равно 'Sales' и значение
столбца "salary" меньше 5000.


Заключение

Команда UPDATE позволяет изменять значения столбцов в существующих
строках таблицы базы данных.

Она полезна для внесения изменений в существующие данные или
исправления ошибок в таблице.
