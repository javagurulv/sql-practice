Бизнес просит вас провести анализ данных хранящихся в базе данных.

Плоскость трейдеров:

1. Найти имена всех трейдеров зарегистрированных в системе.

2. Отсортируйте имена трейдеров в алфавитном порядке.

3. Какие трейдеры не совершили ни одной транзакции за все время работы?

4. Найти для каждого трейдера его транзакцию с наибольшей суммой за все время работы
   и отсортировать список по сумме наибольшей транзакции?

5. Найти для каждого трейдера его транзакцию с наименьшей суммой за все время работы
   и отсортировать список по сумме наименьшей транзакции?

6. Какова средняя сумма транзакций для каждого трейдера за все время работы?

7. Каков общий объем транзакций для каждого трейдера за все время работы?

8. Найти ответы на 4 - 7 для каждого трейдера не за все время,
   а в указанном году (например 2010, или 2021).

9. Стал ли работать лучше (совершать больше транзакций) трейдер
"Jack MA" в разрезе последних 5 лет.

10. Стал ли работать лучше (совершать транзакций на большую общую
сумму) трейдер "Jack MA" в разрезе последних 5 лет.

   все время работы системы?

3. Наибольшая транзакция для каждого города за
   все время работы системы?

4. Наименьшая транзакция для каждого города за
   все время работы системы?

5. Средняя сумма транзакций для каждого города за
   все время работы системы?

6. Каково общее количество трейдеров в каждом городе?

7. Растёт или падает общая сумма транзакций в городе "New York"
   за последние 5 лет? (подготовьте данные для ответа на этот вопрос).

8. Растёт или падает общее количество транзакций в городе "London"
   за последние 5 лет? (подготовьте данные для ответа на этот вопрос).
