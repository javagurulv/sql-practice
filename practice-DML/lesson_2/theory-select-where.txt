Команда SELECT в SQL используется для выборки данных
из таблицы или таблиц базы данных.

Когда мы добавляем простые условия в команду SELECT,
мы можем фильтровать данные и получать только те строки,
которые удовлетворяют определенным условиям.

Пример 1: Выборка всех столбцов из таблицы "users" с определенным условием:
```sql
SELECT *
FROM users
WHERE age > 30;
```
В этом примере мы выбираем все столбцы (`*`) из таблицы "users",
но добавляем условие `WHERE age > 30`, которое фильтрует строки
и выбирает только те, где значение столбца "age" больше 30.
Таким образом, мы получаем только строки пользователей старше 30 лет.


Пример 2: Выборка одного столбца из таблицы "buildings" с определенным условием:
```sql
SELECT building_name
FROM buildings
WHERE capacity > 100;
```
В этом примере мы выбираем только столбец "building_name"
из таблицы "buildings". Добавленное условие `WHERE capacity > 100
фильтрует строки и выбирает только те, где значение столбца
"capacity" больше 100. Таким образом, мы получаем
только названия зданий в которых вместимость больше 100.


Пример 3: Выборка нескольких столбцов из таблицы "products" с определенным условием:
```sql
SELECT product_name, price
FROM products
WHERE category = 'Electronics';
```
В этом примере мы выбираем только столбцы "product_name" и "price"
из таблицы "products". Добавленное условие `WHERE category = 'Electronics'`
фильтрует строки и выбирает только те, где значение столбца
"category" равно "Electronics". Таким образом, мы получаем
только информацию о продуктах из категории "Electronics".


Заключение

Команда SELECT с простыми условиями позволяет фильтровать
данные на основе определенных критериев, чтобы получить только
нужные строки.

Условия могут включать операторы сравнения
(например, `=`, `<`, `>`, `>=`, `<=`, `!=`).

Это позволяет более гибко настраивать выборку данных в соответствии
с требуемыми условиями.
